## 1. 产品概述

基于 Vue 3 + Vite 的网页音乐播放器，支持本地音乐文件管理与在线播放，具备桌面歌词、音频可视化、均衡器等专业功能。目标用户为追求高品质音乐体验的桌面端浏览器用户。

- 解决本地音乐文件在浏览器中的管理与播放需求，填补 Web 端专业音乐播放器的空白
- 以深色沉浸式 UI + 封面主色调动态强调色 + 3D CD 旋转封面打造差异化视觉体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 本地用户 | 无需注册 | 全部功能（本地音乐管理、在线搜索、歌词、可视化等） |

### 2.2 功能模块

1. **音乐库页面**：本地音乐导入、元数据解析、在线搜索、多源管理
2. **歌单页面**：创建/编辑/删除歌单、拖拽排序、封面拼图、m3u 导入导出
3. **搜索页面**：全文搜索、相关度排序、搜索历史
4. **设置页面**：均衡器调节、播放偏好、主题配置
5. **统计页面**：播放历史、听歌时长、收听报告、最常听排行

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 音乐库 | 本地导入 | 选择文件夹扫描 mp3/flac/wav/aac/ogg，URL.createObjectURL 临时访问 |

| 歌单 | 歌单管理 | 创建/编辑/删除歌单，添加/移除歌曲 |
| 歌单 | 排序与封面 | 拖拽排序，前 4 首封面拼九宫格，m3u 导入导出 |
| 搜索 | 全文搜索 | 标题/艺术家/专辑/歌词全文检索，相关度排序，搜索历史 |
| 播放器 | 播放控制 | 播放/暂停/上下首，进度条拖动，音量控制/静音 |
| 播放器 | 播放模式 | 列表循环/单曲循环/随机播放/顺序播放 |
| 播放器 | 高级控制 | 倍速 0.5x-2x，淡入淡出 0.5s 过渡 |
| 播放器 | 迷你模式 | 底部窄栏：封面+标题+播放控制+进度条 |
| 歌词 | LRC 解析 | 按时间戳分行解析，高亮当前行，点击跳转 |
| 歌词 | 桌面歌词 | 独立小窗置顶、背景透明、点击穿透 |

## 3. 核心流程

用户打开应用 → 选择本地文件夹导入音乐 → 自动解析元数据和封面 → 浏览音乐库 → 点击播放 → 封面 3D 旋转 + 主色调提取 → 歌词同步滚动 → 切换可视化模式 → 调节均衡器 → 创建歌单收藏 → 查看收听统计

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色背景（#0a0a0f），动态强调色取自当前播放封面主色调
- **辅助色**：中性灰（zinc 系列）用于文字和边框，强调色高饱和度用于按钮和高亮
- **按钮风格**：圆角微透明玻璃态，hover 时强调色边框发光
- **字体**：标题使用 Outfit（几何感现代字体），正文使用 DM Sans
- **布局风格**：左侧导航栏 + 右侧内容区，底部播放器常驻，响应式适配
- **图标**：Lucide Icons 线条风格
- **封面特效**：圆形 3D 旋转 CD 效果（CSS 3D transform），播放时旋转，暂停时停止

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 音乐库 | 文件导入区 | 拖拽上传区 + 文件夹选择按钮，虚线边框，强调色渐变背景 |
| 音乐库 | 音乐列表 | 虚拟滚动列表，行 hover 高亮，封面缩略图 + 歌曲信息 + 时长 |
| 音乐库 | 在线搜索 | 搜索框 + 结果卡片网格，封面图 + 标题 + 来源标签 |
| 歌单 | 歌单卡片 | 九宫格拼图封面 + 歌单名 + 歌曲数，hover 放大阴影 |
| 歌单 | 歌单详情 | 歌曲列表 + 拖拽手柄 + 删除按钮，顶部大封面 + 信息 |
| 播放器 | 全尺寸播放器 | 居中 3D 旋转 CD 封面 + 歌曲信息 + 进度条 + 控制按钮 + 频谱可视化 |
| 播放器 | 迷你播放器 | 底部窄栏：小封面 + 歌名 + 播放控制 + 迷你进度条 |
| 歌词 | 歌词面板 | 居中滚动歌词，当前行强调色放大高亮，其他行半透明 |
| 歌词 | 桌面歌词 | 独立窗口，透明背景，当前行大字显示，毛玻璃效果 |
| 均衡器 | EQ 面板 | 10 个垂直滑块 + 预设按钮组 + 自定义保存 |
| 可视化 | 频谱面板 | Canvas 绘制，三种模式切换按钮，跟随音乐实时渲染 |
| 搜索 | 搜索页 | 大搜索框 + 搜索历史标签 + 结果分类标签页 |
| 统计 | 收听报告 | 数据卡片 + 图表（柱状图/饼图）+ 排行榜列表 |
| 设置 | 设置页 | 分组设置项，开关/滑块/下拉选择 |

### 4.3 响应式设计

- 桌面优先（≥1280px）：左侧导航 + 右侧内容 + 底部播放器
- 平板（768px-1279px）：底部导航 + 内容区 + 底部播放器
- 手机（<768px）：底部导航 + 全屏内容 + 迷你播放器

### 4.4 动效设计

- 页面切换：fade + slide 过渡
- 播放器封面：CSS 3D rotateY 无限旋转，暂停时 ease-out 停止
- 歌词滚动：smooth scroll，当前行 scale(1.1) + 强调色
- 频谱可视化：requestAnimationFrame 实时渲染，60fps
- 按钮 hover：微弹跳 + 发光边框
- 歌单卡片 hover：translateY(-4px) + box-shadow 增强
- 歌曲切换：淡入淡出 0.5s crossfade
